# ReadWriterProblem
Este proyecto soluciona el problema de lector y escritor enfocado al ámbito de los Sistemas Operativos.

Implementa semáforos que se ven reflejados en la case Read y Writer.

La clase Management.cs se encarga de manejar la sección crítica.

Solo se puede un escritor al tiempo, y pueden estar muchos lectores, pero jamás ambos juntos.
